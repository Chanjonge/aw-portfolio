이미지가 안 보이는 주요 원인:

1. ⭐ Vercel Blob Storage가 생성되지 않음
   → 업로드 실패 또는 상대 경로로 저장

2. 업로드는 되지만 저장 버튼을 안 누름
   → 이제 "저장 버튼을 눌러주세요" 알림 추가됨

3. 기존 이미지가 상대 경로로 저장됨
   → 다시 업로드 필요


【지금 해야 할 일】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1단계: Vercel Blob Storage 생성 ⭐
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

https://vercel.com 접속

→ 프로젝트 선택
→ Storage 탭 클릭
→ "Create Database" 버튼
→ "Blob" 선택
→ 이름: portfolio-uploads
→ Create

→ 자동으로 BLOB_READ_WRITE_TOKEN 추가됨!


2단계: Vercel 배포 완료 대기 (1-2분)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Vercel Dashboard → Deployments → "Ready" 확인


3단계: 이미지 업로드 테스트
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

https://aw-portfolio-chi.vercel.app/admin/login

1. 로그인
2. 포트폴리오 편집 또는 질문 편집
3. 이미지 선택
4. ✅ "이미지 업로드 성공!" 알림 확인
5. ⚠️ 반드시 "저장" 버튼 클릭!
6. 다시 편집하면 이미지가 보여야 함


【개선된 기능】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Before:
  ❌ 업로드 실패 시 명확한 오류 없음
  ❌ 성공 여부를 알 수 없음
  ❌ 저장 버튼을 눌러야 한다는 안내 없음

After:
  ✅ 성공 시: "✅ 이미지 업로드 성공! 저장 버튼을 눌러주세요"
  ✅ 실패 시: "❌ 이미지 업로드 실패: [상세 오류]"
  ✅ 콘솔에 상세 로그 출력


【오류 메시지 해석】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

"BLOB_READ_WRITE_TOKEN is not defined"
→ Blob Storage를 생성하지 않음
→ 1단계 다시 확인!

"파일 업로드에 실패했습니다"
→ 인증 문제 또는 권한 문제
→ 다시 로그인 시도

"인증이 필요합니다"
→ 로그인 만료
→ 다시 로그인


【데이터 확인 방법】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. F12 (개발자 도구)
2. Network 탭
3. 이미지 업로드 시도
4. /api/upload 요청 클릭
5. Response 탭에서 URL 확인

올바른 응답:
{
  "url": "https://xxx.public.blob.vercel-storage.com/uploads/..."
}

잘못된 응답:
{
  "error": "...",
  "details": "..."
}
